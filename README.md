# Web Direction Lab

Webディレクター・Webコンサルを目指す人向けの学習メディアです。

## コンテンツ

- ナレッジ：SEO、Web制作、アクセス解析、システム、AI、マーケティング
- Webニュース：公式発表を「概要・影響・次のアクション」で整理
- 毎日更新：毎朝6:00にナレッジ・Webニュース・ショート動画台本を生成し、レビューPRの承認後に公開

## 開発

```bash
npm install
npm run dev
```

公開用のAPIキーやSNSトークンはリポジトリへ保存せず、GitHub Actions Secretsで管理します。

毎日更新の初回設定と確認手順は [`automation/README.md`](automation/README.md) を参照してください。
